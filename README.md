<h1>Advanced E-commerce App</h1>

<p>
<strong>Advanced E-commerce App</strong> is a fully featured online shopping platform built with a modern frontend and backend architecture.
It provides a complete e-commerce experience including product listings, cart management, and user authentication.
</p>

<p>
🔗 <strong>Live Demo:</strong>
<a href="https://e-commerce-store-smoky-xi.vercel.app" target="_blank">
https://e-commerce-store-smoky-xi.vercel.app
</a>
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>Modern and responsive UI</li>
  <li>Product listing and detail pages</li>
  <li>Shopping cart functionality (Add / Remove items)</li>
  <li>User authentication (Login / Register)</li>
  <li>Scalable and clean project structure</li>
  <li>TypeScript-based frontend</li>
</ul>

<hr>

<h2>🧱 Tech Stack</h2>

<h3>Frontend</h3>
<ul>
  <li>React</li>
  <li>TypeScript</li>
  <li>HTML5 & CSS3</li>
</ul>

<h3>Backend</h3>
<ul>
  <li>Node.js</li>
  <li>Express.js</li>
  <li>RESTful APIs</li>
  <li>Database Integration (MongoDB or SQL)</li>
</ul>

<h2>🛠 Installation</h2>

<h3>1. Clone the Repository</h3>

<pre><code>git clone https://github.com/Mahmoud-Refaat07/Advanced-e-commerce-app.git
cd Advanced-e-commerce-app
</code></pre>

<h3>2. Install Dependencies</h3>

<h4>Frontend</h4>

<pre><code>cd frontend
npm install
</code></pre>

<h4>Backend</h4>

<pre><code>cd backend
npm install
</code></pre>

<hr>

<h2>▶️ Running the Application</h2>

<h3>Start Frontend</h3>

<pre><code>cd frontend
npm start
</code></pre>

<p>App runs by default on: <strong>http://localhost:3000</strong></p>

<h3>Start Backend</h3>

<pre><code>cd backend
npm start
</code></pre>

<p>Ensure your backend server is connected to the database.</p>

<hr>

<h2>⚙️ Environment Variables</h2>

<p>Create a <code>.env</code> file in both frontend and backend directories.</p>

<pre><code># Example Configuration
REACT_APP_API_URL=http://localhost:5000/api
PORT=5000
MONGO_URI=your_database_connection_string
JWT_SECRET=your_secret_key
</code></pre>
